はじめに

このページは C言語ビギナーが、友達に C言語の文法を一通り教えるために作成したものです。
このページ内で掲載しているプログラムは、無断でコピーまたは改変して使用していただいて構いませんが、使用により発生するいかなる損失・損害について、作者は一切責任を負いません。

Hello,world!

C言語の文法について触れる前に、まずは C言語で書かれた簡単なプログラムを動かしてみましょう。

このプログラムは、画面に "Hello,world!" という文字列を出力するだけのものです。
(1) は画面に文字列を出力するためのおまじないのようなものだと思ってください。 (2) の printf関数というものを使用するために必要なので、printf関数を使用する際には忘れずに記述するようにしましょう。
(2) の printf関数は、丸括弧の中に記述されている文字列を出力します。 プログラム内に文字列を記述する際は、プログラム自体の文字列と区別するために " (ダブルクォーテーション) で括ってやる必要があります。 また、文字列の最後に記述されている '\n' は改行を意味している。
